# Custom Telegram Calls

Telegram бот для управления друзьями и инициации голосовых звонков. Приложение предоставляет удобный способ добавлять контакты через уникальные ключи дружбы, управлять списком друзей и совершать защищенные голосовые вызовы в обход региональных блокировок.

## Возможности

- Добавление друзей через уникальный ключ дружбы с системой подтверждения запросов.
- Управление списком друзей с возможностью удаления контактов.
- Инициация голосовых звонков и управление входящими вызовами.
- Система уведомлений о входящих звонках и статусе пользователей.
- Отображение профиля с личной информацией и статусом активности.
- Подробное описание работы системы через интерактивное меню.
- Кеширование данных через Redis для оптимизации производительности.
- Логирование всех операций для отладки и мониторинга.

## Стек

- Node.js с TypeScript
- Telegram Bot API через grammy
- PostgreSQL с Prisma ORM
- Redis для кеширования и управления состоянием
- Pino для логирования
- Prisma для миграций базы данных

## Требования

- Node.js 18+
- PostgreSQL база данных
- Redis сервер
- Telegram Bot Token

## Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone https://github.com/your-username/custom-telegram-calls.git
cd custom-telegram-calls
```

2. Установите зависимости:

```bash
npm install
```

3. Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=your_telegram_bot_token
DATABASE_URL=postgresql://user:password@localhost:5432/telegram_calls
REDIS_HOST=localhost
REDIS_PORT=6379
OWNER_ID=your_telegram_id
LOG_LEVEL=info
```

4. Примените миграции базы данных:

```bash
npm run migrate:dev
```

5. Сгенерируйте Prisma Client:

```bash
npm run generate:prisma
```

6. Запустите бот в режиме разработки:

```bash
npm run dev
```

Или в продакшене:

```bash
npm run build
npm start
```

## Структура проекта

- `src/main.ts` - точка входа приложения и инициализация бота.
- `src/handlers` - обработчики команд и событий Telegram.
- `src/keyboards` - клавиатуры и меню для взаимодействия с пользователем.
- `src/services` - бизнес-логика для работы с дружбой, профилем и уведомлениями.
- `src/database` - конфигурация Prisma и схема данных.
- `src/messages` - константные сообщения и тексты для пользователя.
- `src/lib` - утилиты для логирования, Redis, состояний и пагинации.
- `src/types` - TypeScript типы для всех сущностей приложения.
- `src/utils` - вспомогательные функции для работы с данными.

## Жизненный цикл дружбы

1. **Создание ключа дружбы** — пользователь получает свой уникальный идентификатор.
2. **Отправка запроса** — другой пользователь вводит ключ и отправляет запрос.
3. **Подтверждение** — получатель видит запрос и может принять или отклонить.
4. **Активная дружба** — после принятия пользователи могут совершать видеозвонки.

## Команды бота

- `/start` — начало работы, показ приветственного сообщения.
- Профиль — просмотр личной информации и статуса.
- Друзья — управление списком контактов.
- Как это работает — подробное описание функциональности.
